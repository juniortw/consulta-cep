# Consulta CEP

Site simples para consultar enderecos pelo CEP usando a API publica ViaCEP.

## Como usar

Acesse o site publicado:

https://juniortw.github.io/consulta-cep/

Digite um CEP com 8 numeros e clique em **Buscar**.

Para rodar localmente, abra o arquivo `index.html` no navegador.

## Tecnologias

- HTML
- CSS
- JavaScript
- ViaCEP
- Android WebView

## APK Android

O APK de teste e gerado pelo GitHub Actions no workflow **Build Android APK**.

Depois que o workflow terminar, baixe o artefato `consulta-cep-debug-apk` na aba **Actions** do repositorio.
